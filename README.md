# acs-middleware

Node service to populate device data to database

### Installation

Download code from github
``` sh
    $ git clone https://github.com/MachineCDN/acs-middleware.git
```
Install libraries
``` sh
    $ cd acs-middleware
    $ npm install
```
Config envirenment variables
``` sh
    copy .env.example file and update variables
    rename it .env
```
Run script
``` sh
    $ pm2 start app.js
```
